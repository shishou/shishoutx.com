		"font_size": 14,
		//默认字体大小

		"font_face": "Inconsolata 微软雅黑",

		"line_padding_top": 1,

		"highlight_line": true,

		"rulers": [80,100],

		"translate_tabs_to_spaces": true,
		"tab_size": 2,
		//把Tab对齐转化为空格对齐

		"auto_find_in_selection": true,
		//开启选中范围内搜索（Search in selection）

		"save_on_focus_lost": true,
		//焦点丢失后自动保存

		"word_wrap": false,
		//自动换行

		"draw_white_space": "all",
		//显示Tab、空格

		"show_encoding": true,
		//右下角显示当前文件的编码

		"trim_trailing_white_space_on_save": true,
		//保存时自动去除行末无用空格

		//侧栏文件树文件夹加粗显示。

		//标题栏显示文件全路径

Emmet 快速生成HTML、CSS

2016-03-16 21:59

Emmet DOM：http://docs.emmet.io/
使用Package Control来安装，
Ctrl+Shift+P
键入
emmet
第一个就是,直接回车，安装完成会跳一个说明窗口。

如果你直接使用，会提示你要PyV8,PyV8安装方法如下：
在安装Emmet后按
Ctrl+`
你会看到一个类似 "https://raw.github.com/emmetio/pyv8-binaries/master/pyv8-win64-p3.zip"的链接在里面。
复制链接下载，把解压的文件复制到
"%USERPROFILE%\AppData\Roaming\Sublime Text 3\Installed Packages\PyV8"
（%USERPROFILE% 相当于你的 C:\Users\"当前用户名"）
如果没有这个目录的话就新建一个。
Win+R 打开cmd，复制粘贴以下命令回车
md "%USERPROFILE%\AppData\Roaming\Sublime Text 3\Installed Packages\PyV8"
最终就是这样一个目录
%USERPROFILE%\AppData\Roaming\Sublime Text 3\Installed Packages\PyV8\pyv8-win64-p3\几个py结尾的文件

主要用法可以看这里：Emmet：HTML/CSS代码快速编写神器

CSS Format

2016-03-24 16:26

可以将任意的 CSS、SASS、SCSS、LESS 代码格式化特定格式
用法:
打开 CSS 文件后，可以通过以下任一方法执行格式化命令：

  • 右键菜单：CSS Format >
  • 编辑菜单：Edit > CSS Format >
  • 命令面板：使用快捷键 Ctrl+Shift+P(Windows/Linux) 、Cmd+Shift+P(OS X) 打开命令面板，输入 Format CSS，选择对应格式的菜单回车即可
